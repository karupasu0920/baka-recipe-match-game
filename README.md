# バカレシピ展 応援ゲーム 〜缶バッジペア消し〜

野島慎一郎さんの **「バカレシピ展」** を応援するために作られた、  
神経衰弱スタイルのペアマッチングゲームです🎪

> **📌 展示情報：** 2026年4月29日 – 5月3日 / 清瀬市郷土博物館にて開催

---

## 🎮 概要

22枚の缶バッジカードが画面にランダム配置されます。  
同じ絵柄のバッジを2枚タップして消していき、**45秒以内に全ペアを消せばクリア！**  
連続マッチで「コンボボーナス」が発動し、タイムが短縮されます。

さらに、**シークレット缶バッジ** 🌻 が隠れており、  
マッチさせると清瀬市の特別演出が表示されます。

---

## 🕹️ 遊び方

1. **GAME START** を押してゲーム開始
2. 同じ絵柄のバッジを2枚タップ → ペアが消える
3. 全11ペア（22枚）を消すとクリア！
4. タイムに応じてバカレシピ風のランクが判定される
5. ハンドルネームを入力してランキングに登録 & Xでシェア

## 🔥 コンボボーナス

**3連以上の連続マッチ**でタイム短縮が発動します。

| コンボ数 | タイム短縮 |
|---|---|
| 1〜2連 | なし（演出のみ） |
| 3連 | −0.10秒 |
| 4連 | −0.15秒 |
| 5連以上 | −0.30秒 / ペア |

**フルコンボ理論最大短縮：2.35秒**

> 💡 1〜2連は演出のみ（「○連勝！」バナー・SEは鳴ります）。  
> **3連からボーナス発動**、**5連以降が本当の実力者ゾーン**です。  
> どん二郎マスター（〜6秒）達成には**コンボを最後まで繋ぎ切る**のが鍵。
